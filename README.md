# Portfolio - Sahrul Rafi Zulfitra ‚úå <a name="about-project"></a>

This Personal Portfolio is a high-performance Progressive Web App (PWA) crafted to showcase my journey as a Software Engineer. Built with React and styled with Tailwind CSS, it emphasizes speed, interactivity, and modern design principles.

The site delivers a seamless user experience using React Router 6 for navigation and GSAP for immersive, fluid animations. It is engineered for optimal performance and security, deployed on Vercel with Cloudflare CDN integration.
